# OpenMemoryUI Brag Composition Design

- Canvas: 1920x1080 landscape, 21.5 seconds.
- Palette: project dark #0d141f / #121b29, text #e8eef5, lines #263548, accents #6fd3ff, #ffb454, #a78bfa, #57e6c4.
- Typography: Bricolage Grotesque for display, Atkinson Hyperlegible for UI copy, Spline Sans Mono for labels.
- Motion: restrained app-store launch motion. Fast entrances, readable holds, clean exits.
- Product truth: recreate the actual Memory Glassbox UI and show the message -> pipeline -> memory write flow.
